# Chat App

A real-time chat app built with vanilla HTML/CSS/JS and Supabase.

## Features
- Random username on every visit
- Real-time messages across all devices
- Typing indicators
- Optimistic UI

## Tech stack
- Frontend: HTML, CSS, vanilla JS
- Backend: Supabase (Postgres + Realtime)
- Hosting: Cloudflare Pages

## Local development
1. Clone the repo
2. Open `src/index.html` with Live Server in VS Code
3. Start chatting
